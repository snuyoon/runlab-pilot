# 아이폰 실기기 테스트 체크리스트

> 배포 URL: **https://runlab-pilot.vercel.app**
> 관리자: **https://runlab-pilot.vercel.app/admin** (관리자 키 필요)
> 파일럿 배포 전, 연구자 본인 아이폰에서 아래 순서로 1회 이상 통과 확인.

## 0. 설치
- [ ] Safari로 URL 접속 → 공유 버튼 → **홈 화면에 추가**
- [ ] 홈 화면 아이콘(보라색 R)으로 실행 시 주소창 없는 전체 화면으로 열림
- [ ] 노치/하단 홈바에 콘텐츠가 가려지지 않음

## 1. 로그인 (사전 등록 코드)
- [ ] 아무 코드(`ABC-123`) 입력 → "등록되지 않은 참여 코드" 거부됨
- [ ] `TEST-01` 입력 → 홈 진입, 헤더에 "TEST-01 (연구자 테스트)" 표시
- [ ] 앱 종료 후 재실행 시 로그인 유지(코드 재입력 불필요)

## 1-b. 자동 수집 + 관리자 확인
- [ ] 아이폰에서 기상 설문/RPE 제출 → PC에서 /admin 열어 관리자 키 입력 → 격자에 즉시 반영되는지
- [ ] 비행기 모드에서 RPE 제출 → 다시 온라인 → 앱 재진입 → /admin에 나타나는지 (오프라인 큐 재시도)
- [ ] /admin에서 전체 CSV 다운로드 열어보기

## 2. 주간 OSTRC 팝업 + 설문
- [ ] 첫 홈 진입 시 "주간 건강 설문 시간!" 팝업이 뜸 (이번 주 미완료 상태이므로)
- [ ] "나중에 하기" → 앱을 완전히 껐다 다시 열면 팝업이 다시 뜸
- [ ] 설문 완료(문제 없음 경로: 4문항 모두 첫 번째 선택) → 홈에서 "이번 주 완료!" 표시, 팝업 안 뜸
- [ ] 부상 경로: Q1~Q4 문제 있음 → 부상 → 부위 선택 → 다른 문제 "예" 반복 동작 확인
- [ ] 긴 목록(부상 부위, MH-1)에서 스크롤 정상

## 3. 알람 + 수면 + 기상 설문 (핵심!)
- [ ] 알람 설정에서 2~3분 뒤 시각으로 설정
- [ ] "취침 시작하기" → 수면 화면(어두운 시계) 진입
- [ ] **무음 모드 스위치 켠 상태에서도 알람음이 나는지** ← iOS에서 WebAudio는 무음 스위치에 묶이는 경우 있음. 안 나면 "무음 모드 해제 안내"를 취침 화면에 추가해야 함
- [ ] 화면이 자동으로 꺼지지 않는지 (Wake Lock) — 5분 이상 방치 확인
- [ ] 설정 시각에 알람음 + 화면 전환
- [ ] 밀어서 끄기 → 기상 설문 자동 진입 → 제출 → 홈에 ✅
- [ ] (실사용 조건) 충전기 연결 + 밝기 최저로 하룻밤 실제 테스트

## 4. 러닝 세션 RPE
- [ ] 홈 → 러닝 세션 기록 → 1~10 선택 시 라벨 표시 → 기록
- [ ] 같은 날 2번째 기록 가능 ("오늘 2건 기록됨")
- [ ] 숫자 입력 시 화면 확대(줌)가 일어나지 않음

## 5. 데이터 내보내기
- [ ] 내 기록 → 통계/최근 기록 정상 표시
- [ ] "연구자에게 데이터 보내기" → iOS 공유 시트(카톡/메일/에어드랍)로 JSON 전송됨
- [ ] 받은 JSON 파일 열어서 응답 값 확인

## 6. 알려진 제약 (참여자 안내문에 포함할 것)
- 알람은 **화면을 켠 채로 충전하며 자야** 울림 (Safari 백그라운드 알람 불가)
- 브라우저 데이터 삭제(방문 기록 지우기) 시 기록 유실 → 주 1회 데이터 전송 권장
- 홈 화면 앱과 Safari 탭은 저장소가 분리됨 → **반드시 홈 화면 아이콘으로만 사용**하도록 안내
