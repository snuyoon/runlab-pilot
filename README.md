# RunLab Pilot — 러닝 연구 파일럿 참여자 앱

> AI 스마트 러닝워치 연구의 **파일럿 테스트(약 10명)** 용 참여자 앱.
> 기존 컨셉 데모(runlab-demo)에서 게이미피케이션(동물 키우기)을 제거하고,
> 알람 + 설문 수집 기능을 연구용으로 강화한 버전입니다. **아이폰 전용**으로 최적화되어 있습니다.

---

## 참여자가 하는 일

| 주기 | 할 일 | 화면 |
|---|---|---|
| 매일 밤 | 워치 착용 → 취침 시작 (화면 켠 채 충전) | `/sleep` |
| 매일 아침 | 알람 해제 → **기상 설문** 자동 실행 (수면질/피로/기분, 1~5점) | `/ema` |
| 러닝 후 | **세션 강도(RPE)** 기록 (1~10점 + 시간/메모) | `/rpe` |
| 매주 월요일 | **OSTRC-H2 주간 건강 설문** — 앱 진입 시 팝업으로 알림 | `/ostrc` |

## 주요 기능

### 1. 알람 + 수면 플로우 (`/alarm`, `/sleep`)
- 기상 알람/취침 리마인더 시간 설정 (휠 UI)
- 취침 시작 → 수면 화면(어두운 시계) → **설정 시각에 자동 알람** (WebAudio 알람음)
- Wake Lock API로 화면 꺼짐 방지. **충전기 연결 + 화면 켠 채로 두는 것이 전제** (웹앱 제약)
- 밀어서 알람 끄기 → 기상 설문 자동 진입, 취침/기상 시각 로그 저장

### 2. 기상 EMA 설문 (`/ema`)
- 알람 해제 직후 자동 실행되는 3문항(수면의 질, 근육통/피로감, 기분), 이모지 1~5점

### 3. 러닝 세션 RPE (`/rpe`)
- "이번 세션은 얼마나 힘드셨나요?" 1~10점 (Borg CR-10 세션 RPE)
- 러닝 시간(분), 메모 선택 입력. 하루 여러 세션 기록 가능

### 4. 주간 OSTRC-H2 설문 (`/ostrc`)
- IOC modified OSTRC-H2 **한국어판** (KSOC 번역·문화적 적응 최종판) 문항 사용
- 매주 월요일부터 완료 전까지 홈 진입 시 팝업 알림
- 분기 구조: 핵심 4문항(0/8/17/25점, 심각도 0~100) → 문제 유형(부상/질병/정신건강)
  → 부상 부위 / 질병 증상군 / 정신건강 MH-1~6 → 추가 문제 반복 등록
- 문항 데이터: [src/data/ostrc.ts](src/data/ostrc.ts)

### 5. 내 기록 + 데이터 수거 (`/dashboard`)
- 응답 요약, 최근 기록 확인
- **JSON/CSV 내보내기**: 파일럿에서는 데이터가 참여자 기기(localStorage)에 저장되므로,
  "연구자에게 데이터 보내기" 버튼(iOS 공유 시트)으로 카톡/메일/에어드랍 수거

## 파일럿 운영 방법 (아이폰)

1. Safari로 배포 URL 접속 → 공유 → **홈 화면에 추가** (전체 화면 PWA로 실행됨)
2. 참여 코드 입력 (참여자별 고유 코드, 예: `SNU-001`)
3. 알람 시간 설정 후 매일 취침 시 "취침 시작하기"
   - **중요**: 충전기에 연결하고 화면을 켠 채로 머리맡에 둬야 알람이 울림 (웹앱은 백그라운드 알람 불가)
4. 참여 종료 시(또는 주 1회) 내 기록 → "연구자에게 데이터 보내기"

## 기술 구조

- Next.js 16 (App Router) + React 19 + TypeScript + Tailwind v4 + Framer Motion
- 상태/데이터: [src/store/studyStore.ts](src/store/studyStore.ts) — 모든 읽기/쓰기 단일 창구.
  현재 localStorage, 이후 서버(DB) 연동 시 이 파일 내부만 교체하면 됨
- 저장 데이터: 설정(참여코드/알람), 기상 EMA, 세션 RPE, OSTRC 주간 응답, 수면 로그
- 배포: Vercel (`npm run build` 정적 생성)

```bash
npm install
npm run dev    # http://localhost:3000
npm run build  # 프로덕션 빌드
```

## 알려진 제약 (웹앱)

- **백그라운드 알람 불가**: Safari가 닫히거나 화면이 꺼지면 알람이 울리지 않음 → 수면 화면을 켠 채로 사용
- 푸시 알림 없음: OSTRC 알림은 "앱을 열었을 때 팝업" 방식
- 데이터는 기기별 저장: 브라우저 데이터 삭제 시 유실 → 주기적 내보내기 권장
- 위 제약이 파일럿에서 문제가 되면 → 서버(Supabase 등) + iOS 웹푸시(홈 화면 설치 시 지원) 또는 네이티브 전환 검토
